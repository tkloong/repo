Poisson Image Cloning

Poisson Image Cloning is a gradient-based image cloning. It preserve the gradient of the image and cloning the color from the boundary to interior. Unfortunately, it has to solve a large sparse linear system, poisson equation, thus it consume a lot of time.

MVC Image Cloning

The algorithm is easy and yield much similar result compare with poisson image cloning. The most advantages of using this approach is it construct a closed form formula to interpolate between source patch and target patch. Hence eliminate the need of solving large linear system.

Optimization

Instead of directly solving a large sparse linear system, multigrid method is used to solve a hierarchical of differential equation. Concretely, we first sample down the target image, solve these small linear system by Jacobi method, scale up the image, initialize with the solution of small linear system and solve the higher hierarchy linear system. By using GPU, the speedup is significant. For example, a 1024x639 pixel of target patch is seemlessly clone in average 0.001525s. The cloning result is below.
